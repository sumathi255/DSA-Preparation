# Armstrong Number Check (Basic Python)

## What is an Armstrong Number?

An **Armstrong number** is a number that is equal to the sum of its digits raised to the power of the number of digits.

### Example:
- **153**
  - Digits = 3
  - 1³ + 5³ + 3³ = 1 + 125 + 27 = **153**
  - ✔ Armstrong Number

- **123**
  - 1³ + 2³ + 3³ = 36
  - ✘ Not an Armstrong Number

---

Explanation (Very Simple)
Step 1: Count Digits
We count how many digits are present in the number.

Step 2: Power Calculation
Each digit is raised to the power of the number of digits and added.

Step 3: Comparison
If the final sum equals the original number, it is an Armstrong number.

Sample Input / Output
Input
Enter a number: 370
Output
Armstrong number
